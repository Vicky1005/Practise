1. Открыть gitBash
2. Ввести git gui&
//появляется окно GUI
кнопка rescan помогает синхронизировать состояние файлов и посмотреть что изменилось
3. Кликаем на иконку файла в разделе Unstaged Changes
//файл попадает в Staged Changes/will commit  - то что пойдет в комит
если еще раз нажать на иконку  - перебросится в unstaged
4. Пишем комит сообщение внизу и нажимаем комит
в GitBAsh git log можем проверить что комит успешно создан (последние комиты отображаются сверху)
5. Push появляется окно push
Можно закомитать несколько строчек (не обязательно файл) - просто выделить необходимые строки справа на панели 
в GUI и далее нажать кнопку файла и комит


Инструмент gitk
Запустить гитбаш
написать gitk&
появляетя табличка
он нужен для просмотра истории
