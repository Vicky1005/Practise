1. Создать файл .gitignore
Комитать нельхя бинарный файлы или файлы которые не несут смысловой нагрузки для исходного кода проекта 
(скриншоты, файлы пользовательских настроек).
Позволяет создать маску файла и забыть о нем.

Синтаксис
# no log files:
*.log

#but do track error.log, even though you're ignoring .log files above:
!error.log

#only ignore the TODO file in the current directory not subdir/TODO
/TODO

#ignore all files in the build / directory
build/

#ignore doc/notes.txt, but not doc/server/arch.txt
doc/*.txt

#ignre all .pdf files in the doc/ directory
doc/**/*.pdf
